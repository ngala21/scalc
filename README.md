# Scalc

#### By **Ngala Mike**

## Description
Calculator that performs simple arithmetic operations. 

## Setup/Installation Requirements
- Open your terminal.
- Initialize git on your terminal.
- git clone <https://github.com/ngala21/scalc.git>.
- open 'scalc' folder.
- Run it on a browser preferably Chrome.
